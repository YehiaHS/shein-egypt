# SHEIN Egypt — Fashion Collection 2026

An interactive HTML presentation for the SHEIN 2026 Egypt Fashion Collection.

🔗 **Live:** https://yehiahs.github.io/shein-egypt/

## Features
- 12 slides with smooth transitions
- Real Three.js 3D billboard models for campaign slides
- Animated particle background
- Keyboard, mouse wheel, swipe, and click navigation
- All images embedded (single self-contained HTML file)
